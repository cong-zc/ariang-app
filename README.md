# ariang-app
aria2c下载器python版，用pywebview写了ui，使用的是ariaNG的前端


# 使用方法：
### 安装依赖包：
```bash
pip install -r requirements.txt
```
### 运行程序：  
1. 在命令行中输入`python main.py`
### 打包程序：  
1. 通过cmd进入项目目录，运行build.bat生成exe文件。
2. 运行.exe文件即可。
